
命令格式：
```
du [OPTION]... [FILE]...
du [OPTION]... --files0-from=F
```

OPTION 参数
```
-h, --human-readable
    以K，M，G为单位，显示文件的大小

-s, --summarize
    只显示总计的文件大小

-S, --separate-dirs
    显示时并不含其子文件夹的大小

-d, --max-depth=N
    显示子文件夹的深度（层级）
```
