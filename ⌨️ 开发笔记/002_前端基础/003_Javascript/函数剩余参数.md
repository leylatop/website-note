```javascript
function func(...args) {
	console.log(args)
}
```

剩余参数语法，不定数量的参数表示为一个数组，由... + 变量名args组成
- ...args 表示为不确定有多少个参数
- args为一个数组


场景：如果函数的最后一个命名参数以...为前缀，则它将成为一个由剩余参数组成的真数组

剩余参数和arguments对象的区别

- 剩余参数只包含没有形参的实参，arguments为所有实参

* 剩余参数的类型是数组类型，arguments为类数组
* arguments对象上还有一些附加的属性

  

推荐使用剩余参数，也叫rest参数
